# Mangrove

Mangrove is a Node.js ORM(Object-Relational Mapping) tool for Postgres and MySQL.

Mangrove follows Semantic Versioning and supports Node v10 and above.

## Built With

- JavaScript
- TypeScript
- Shell


## Installation

```
$ npm install mangrove

```
